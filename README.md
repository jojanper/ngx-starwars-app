# ngx-starwars-app

> Simple Angular based frontend client for [the Star Wars API](https://swapi.co/)

## Quickstart

### Install dependencies
```
npm i
```

### Start (development) server
```
npm start
```
Open http://localhost:3002 in your browser.

### Run unit tests
```
npm test
```
